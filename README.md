## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hanzlashahzad01/Valentines-Day-Website.git
   ```

   ```bash
   cd Valentines-Day-Website
   ```

   ```bash
   npm run dev
   ```

   ```bash
   npm run build
   ```
